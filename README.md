# Vaibhav's Portfolio

A personal portfolio website built with pure HTML, CSS & JavaScript.

## 🚀 Deployment

This project is deployed on **Vercel** as a static site.

## 📁 Structure

```
Vaibhav_portfolio/
├── index.html      # Main portfolio file (single-page)
├── vercel.json     # Vercel deployment configuration
├── .gitignore
└── README.md
```

## 🛠️ Local Development

Simply open `index.html` in your browser, or use a local server:

```bash
npx serve .
```
